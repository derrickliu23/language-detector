How it works

STEP 01
Extract n-grams
Your text is sliced into overlapping character windows of size 1–4. "hello" produces h, he, hel, hell, e, el… and so on.
STEP 02
Rank by frequency
N-grams are counted and sorted most→least common. This produces a ranked fingerprint — not a bag of words, just an ordered frequency list.
STEP 03
Compare to profiles
Each language's profile was built the same way, from sample text. We measure how far your ranking is from each profile using out-of-place distance.
STEP 04
Pick the winner
The language with the smallest distance (most similar fingerprint) wins. Google does this with gigabytes of training data; we use a few paragraphs per language.
